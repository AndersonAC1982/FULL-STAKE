Principais termos do glossario de Cloud Computing

Cloud Computing
É a entrega de serviços de computação pela internet.
BaaS
Backup como Serviço, que oferece soluções de backup e recuperação de dados na nuvem.
IaaS
Infraestrutura como Serviço, que fornece recursos de computação virtualizados pela internet.
PaaS
Plataforma como Serviço, que oferece uma plataforma para desenvolvimento, teste e implantação de aplicativos.
SaaS
Software como Serviço, que disponibiliza software via internet, sem necessidade de instalação local.
